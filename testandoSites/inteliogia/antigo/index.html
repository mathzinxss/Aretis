Dúvidas Frequentes

Como vou ter acesso ao treinamento?

Assim que a sua compra for aprovada, será enviado um link para o e-mail cadastrado. Lá você poderá criar uma senha e assim conseguirá ter acesso às aulas do treinamento.

Por quanto tempo terei acesso ao treinamento?

O acesso a este treinamento é VITALÍCIO. Isso significa que você terá acesso as aulas atuais e futuras atualizações para sempre (enquanto durar o suporte) e sem valor adicional.

Todas as aulas já estão disponíveis?

Assim que sua compra for aprovada, você terá acesso ao conteúdo que está dentro do treinamento. Atualmente já temos mais de 200 aulas publicadas, mas a cada semana sai um módulo novo.

Qual a garantia do Treinamento Modo Front-end?

Não curtiu o conteúdo? Sem problema! Você pode solicitar o reembolso dentro de um período de 7 dias após a compra*, ou seja, você poderá assistir algumas aulas e se não gostar pode pedir 100% do seu dinheiro de volta.

Irei aprender back-end neste treinamento?

Como o próprio já diz, este treinamento é voltado apenas para o front-end. Vamos aprender HTML, CSS e JavaScript. Ainda não temos planos de inserir back-end neste treinamento.

Este treinamento possui certificado?

Sim! Ao finalizar este curso você poderá emitir o certificado. Ele é válido em todo território nacional e poderá ser emitido de forma online em PDF (Lei 9394/96, Decreto 5154, Deliberação CEE 14/97).

Terei suporte ao longo do curso?

Sim! Você terá um canal exclusivo para tirar suas dúvidas.

Qual a carga horária deste treinamento?

Atualmente a carga horária é de 22 horas, mas esse valor aumenta a medida que inserimos novos conteúdos todas as semanas

Preciso ter algum conhecimento para conseguir realizar o Treinamento Modo Front-end?

Não! O Treinamento Modo Front-end foi pensado para quem tem pouco ou 0 conhecimento sobre HTML, CSS e JavaScript. Portanto, mesmo que você nunca tenha feito uma única linha de código, você vai conseguir realizar o treinamento.

É garantido que terei obtenção de lucros?

Não! Infelizmente isso não podemos garantir. Esteja ciente que você terá aulas focadas no seu desenvolvimento e se você se dedicar de verdade, com toda certeza irá aprender a criar sites, mas o que você fará com esse conhecimento depende apenas de você. Por isso não garantimos obtenção de lucros.
